## LifeOS 是个什么东西？

简单说，LifeOS 是跑在 Obsidian 上的一套插件+模板组合。它把知识管理、日程规划、任务跟踪这些事情整合进了一个仓库里，开箱即用，不用自己从头搭架子。

它的核心逻辑是 PARA 方法——把所有内容按"项目、领域、资源、存档"四个维度分类，配合日记/周记/月记/年记的周期笔记体系，帮你把散落各处的想法和任务理顺。

## 专业版多了什么？差距在哪里？

专业版（Pro）在免费版的基础上多了几个关键功能：

**大日历日程规划** ——一个可以拖拽的全局日历视图，支持日/周/月切换。任务可以直接拖进日历里规划截止时间，这个体验比单纯的文字清单直观太多了。

**远程日历订阅** ——支持 iCloud、飞书、滴答清单等 ics/CalDAV 协议的日历，把所有日程集中到一个地方管。iPhone 用户特别实用，可以直接双向同步提醒。

**高级列表** ——解决了 Obsidian 里任务/笔记越来越多之后难以找到的问题，支持搜索、排序、过滤、翻页，快速找到散落各处的任务和文件。

**主页视图** ——一个仪表盘式的首页，一眼看清系统的输入、项目进展、任务完成情况。

**主题笔记** ——PARA 升级版，降低了新手上手门槛，只需关注"输入、输出、主题"三件事。

这几个功能里，大日历和高级列表是让效率感提升最明显的两个。如果你的任务量上来了、日程协调变复杂了，这两样东西会让你明显感觉更"可控"。
